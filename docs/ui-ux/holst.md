# Holst — доска прототипов

Ссылка на доску: https://app.holst.so/board/9d7e1040-b138-4fdb-9b2c-7185cedc547f

Если доска приватная, дайте доступ на просмотр участникам проекта.

## Экспорт и хранение

- Используйте экспорт из Holst (PNG/SVG/PDF) только при необходимости для PR/доков.
- Постоянное хранение графики в репозитории не требуется; достаточно ссылки на Holst.

## Привязка к маршрутам

- Student — Lab (Stepper) → /s/labs/:labId
- Student — Submission → /s/submissions/:id
- Teacher — Submissions Table → /t/courses/:courseId/labs/:labId/submissions
- Teacher — Submission (AI) → /t/submissions/:id

## Замечания

- Не выкладывайте приватные данные на доску (PII). Для секретов используйте заглушки.
- Для публичной демонстрации используйте view‑only ссылку.
